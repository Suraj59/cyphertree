## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Latest node and npm is recommeneded, please Run:
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/Suraj59/cyphertree
```
2. Install NPM packages
```sh
npm install
```
3. Run Locally
```sh
npm start
```
